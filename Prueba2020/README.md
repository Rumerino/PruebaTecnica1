# PruebaTecnica1
Arquitectura Comit original:

Prueba2020 recibe el nombre del propio word enviado, sus capas en DDD SOLID son:

Prueba2020.Core: en esta capa se encontrara el core de la aplicacion, en este caso, contendra vistas y controlladores.

Prueba2020.Service: en esta capa se encontraran los servicios como las herramientas necesarias como la conexion entity framework para dichos servicios.
                    Aunque entity al ser conexion de datos y deberia residir en la capa web, por ahorrar tiempo sera incorporada en la capa servicio

Prueba2020.Web: En esta capa se encontraran los pogos y lo necesario para los mismos.
